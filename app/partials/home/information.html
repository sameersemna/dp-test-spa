Sameer Shemna's SPA for Senior Frontend Developers' test

Libraries used:

  • AngularJS
  • RequireJS
  • Highcharts
  • CSS Preprocessor used: LESS.

Additions to AngularJS used:

  • RESTangular
  • Angular Deckgrid
  • HighchartsNG

Requirements completed:

  •  4+ Charts
  •  All charts should be in thumbnail mode on the initial page
  •  All charts should be given a full view mode (lightbox or new page) done via Routes (Refreshable)
  •  All charts should be able to switch something simple, such as the chart type or chart name. Persistent data isn't expected to be implemented if you're using JSON files
  •  The chart data should be pulled from a resource. A simple .json file would suffice
  •  Menu

Additional to requirements:

  •  Close Chart
  •  Add new Chart

I am someone who is in love with his work, especially developing SPAs with Visual Data Representation using Graphs and Charts.

I have not used any CSS Framework like Bootstrap,etc. instead tried to give the dashboard a kind of jQuery masonry makeover, used routes with parameters for Charts lightbox so as to persist on a browser hard refresh.

Visual Data Representation is done using HighCharts, more magic could be added using SVG Javascript Libraries like Raphael and d3.js
